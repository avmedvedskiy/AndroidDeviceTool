# com.avmedvedskiy.scrspy

Пакет для Unity Editor для работы с Android-устройствами:
- установка APK на подключенные устройства через `adb`
- запуск `scrcpy` и запись сессии
- сбор логов устройства
- сохранение артефактов сессии в отдельную папку

## Пункты меню

- `Tools/BuildTools/Android/Open Devices Window`
- `Tools/BuildTools/Android/Install Selected APK And Run scrcpy`

## Что умеет окно Devices

- показывает подключенные Android-устройства
- устанавливает выбранный APK на отмеченные устройства
- запускает и останавливает сессию для конкретного устройства
- делает скриншоты в папку текущей сессии
- открывает папку с логами и артефактами
- позволяет выбрать `Bundle Name` из списка
- открывает окно `Edit` для управления списком bundle names

## Bundle Name

В окне используется только выпадающий список `Bundle Name`.

- `ALL` означает режим без фильтрации по package name
- остальные значения берутся из сохраненного списка
- кнопка `Edit` открывает отдельное окно, где можно:
- добавить новый bundle name
- удалить существующий bundle name
- закрыть окно редактора

При установке APK package name автоматически читается из APK, добавляется в список и выбирается в dropdown.

Текущее выбранное значение сохраняется в `EditorPrefs`.
Список bundle names сохраняется в `ProjectSettings/ScrspySettings.json`.

## Основной сценарий

1. Откройте `Devices Window`.
2. Выберите APK, если нужен сценарий установки.
3. При необходимости выберите `Bundle Name` в dropdown или отредактируйте список через `Edit`.
4. Нажмите `Install APK`, если нужно установить сборку на выбранные устройства.
5. Запустите сессию кнопкой `Start Session`.
6. Завершите сессию кнопкой `Stop Session` или просто закройте окно `scrcpy`.
7. После остановки автоматически откроется папка с логами и артефактами сессии.

## Артефакты сессии

Данные сессии сохраняются в:

`<Application.persistentDataPath>/SessionCaptures/<DeviceName>/<yyyy-MM-dd_HH-mm-ss>/`

В папке сессии могут появляться файлы:
- `log.txt`
- `log_unity.txt`
- `video.mp4`
- `summary.txt`
- `screenshot*.png`

## Поведение сессии

- `scrcpy` запускается в режиме записи видео
- если окно `scrcpy` закрыто вручную, сессия автоматически финализируется
- при завершении сессии останавливается `adb logcat`
- после завершения создается `summary.txt`

## Требования

- Unity Editor `2021.3+`
- Android-устройство с включенной USB-отладкой
- `adb` и `scrcpy` из папки `scrspy` внутри пакета
